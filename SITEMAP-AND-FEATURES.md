## Section 3: Features & Integrations

### Core Features

#### 3.1 WhatsApp Business API Integration
- **Floating WhatsApp Button** on all pages (bottom-right)
  - Pre-filled message templates based on page context:
    - Product pages: "Hi, I'm interested in [Product Name]"
    - Contact page: "Hi, I'd like to request a quotation"
    - General: "Hi, I have a question about Crown Plastic products"
- **Click-to-WhatsApp CTAs** on product pages, RFQ forms, contact sections
- Sends to business WhatsApp number (e.g., +971 XX XXX XXXX)

#### 3.2 Contact Forms & RFQ System
- **General Contact Form** (Contact page)
  - Fields: Name, Email, Phone, Company, Message
  - reCAPTCHA v3 spam protection
  - Email notifications sent to Gmail/Google Workspace (via Nodemailer or Resend API)
  - Auto-reply confirmation email to user

- **RFQ (Request for Quotation) Form**
  - Extended fields: Product selection, Quantity, Delivery location, Project timeline
  - File upload for project drawings/specs (up to 10MB)
  - Form data sent via email to sales team
  - Optional: Store submissions in database for CRM tracking

- **Quick Enquiry Forms** on product detail pages
  - Compact sidebar form: Name, Email, Phone, Message
  - Pre-populated with product SKU

#### 3.3 Downloads Centre
- **Centralized Document Repository**
  - Catalogues (full & product family-specific PDFs)
  - Datasheets (individual product PDFs)
  - Installation guides (PDF/video)
  - Certificates of compliance (PDF)
  - CAD drawings (DWG, DXF formats)
  - 3D models (STEP, IGES) [optional]

- **Advanced Search & Filter**
  - Keyword search (searches file names and metadata)
  - Filters:
    - Product Family (multi-select dropdown)
    - Document Type (Catalogue, Datasheet, Installation Guide, Certificate, CAD)
    - Standard (ISO, DIN, BS EN, ASTM)
    - File Format (PDF, DWG, STEP, etc.)
  - Filter combinations (e.g., "UPVC Pressure + ISO 4422 + Datasheet")

- **Download Tracking** (optional)
  - Capture email before download for marketing (optional popup)
  - Analytics: track most downloaded documents

#### 3.4 News/Blog Module (Optional)
- **CMS-like Blog Functionality**
  - Admin can create/edit articles via CMS (if headless CMS used) or markdown files (if static)
  - Categories: Product Launches, Projects, Company News, Industry Insights
  - Tag system for cross-referencing
  - SEO fields: meta title, description, OG image

- **Article Features**
  - Rich text editor support (headings, lists, images, embedded videos)
  - Featured image
  - Related articles suggestions
  - Social sharing buttons (LinkedIn, Twitter, Facebook, Email)

- **Listing Page**
  - Filter by category/tag
  - Search functionality
  - Pagination or infinite scroll

#### 3.5 Media & Video Section (Optional)
- **Video Library**
  - Product explainer videos (AI-generated with voiceover)
  - Installation tutorial videos
  - Project showcase videos
  - Company overview video

- **AI Video Generation Integration** (Optional)
  - Use tools like Synthesia, HeyGen, or D-ID for AI avatars
  - Or AI voiceover tools (ElevenLabs, Murf.ai) for narrating slideshows
  - Videos embedded on:
    - Home page hero
    - Product category pages
    - Applications/Solutions pages
    - Dedicated Media Gallery page

- **Video Embedding**
  - YouTube/Vimeo embeds
  - Self-hosted videos (via Cloudflare Stream or AWS S3 + CloudFront)
  - Lazy loading for performance

#### 3.6 SEO & Technical Features
- **On-Page SEO**
  - Unique meta titles and descriptions for every page
  - Semantic HTML5 structure
  - Heading hierarchy (H1, H2, H3)
  - Alt text for all images
  - Clean, keyword-rich URLs (e.g., `/products/upvc-pressure-pipes`, `/applications/water-supply`)

  - Next.js Image component for automatic optimization
  - Lazy loading for images and videos
  - Code splitting (automatic with Next.js)
  - CDN integration (Vercel Edge Network or Cloudflare)
  - Lighthouse score target: 90+ on Performance, Accessibility, Best Practices, SEO

## Section 4: Components & Templates (Next.js Architecture)

### 4.1 Reusable Components

#### Layout Components
1. **`<LayoutHeader />`**
   - Logo, main navigation, search bar, language switcher
   - Mobile-responsive hamburger menu

2. **`<LayoutFooter />`**
   - Footer links (Products, About, Resources, Contact)
   - Social media icons
   - Newsletter signup form (optional)
   - Copyright notice, Privacy Policy, Terms links

3. **`<MainLayout />`**
   - Wraps all pages with Header and Footer
   - WhatsApp floating button
   - Scroll-to-top button

#### Navigation Components
4. **`<MainNav />`**
   - Desktop navigation with dropdown menus (mega menu for Products)
   - Hover effects, active states

5. **`<MobileNav />`**
   - Mobile hamburger menu
   - Accordion-style sub-menus

6. **`<Breadcrumbs />`**
   - Dynamic breadcrumb trail (Home > Products > UPVC Pressure Pipes > Product Name)

#### Content Components
7. **`<HeroSection />`**
   - Variants:
     - Home hero (full-screen with video background)
     - Page hero (image + title + breadcrumbs)
   - Props: title, subtitle, backgroundImage/Video, CTAs

8. **`<ProductFamilyCard />`**
   - Used on Home and Products landing page
   - Props: icon, title, description, link
   - Hover animation

9. **`<ProductCard />`**
   - Used on category pages and search results
   - Props: image, productName, SKU, shortDescription, link
   - "View Details" and "Quick Enquiry" buttons

10. **`<SpecsTable />`**
    - Displays technical data tables from catalogue
    - Props: tableData (array of objects), headers
    - Responsive (horizontal scroll on mobile)
    - Sortable columns (optional)

11. **`<StandardsAccordion />`**
    - Accordion UI for listing standards (ISO, DIN, BS EN, ASTM)
    - Each item: standard name, description, which products comply
    - Expand/collapse animation

12. **`<DownloadCard />`**
    - Used in Downloads Centre
    - Props: documentTitle, fileType, fileSize, thumbnail, downloadURL
    - Download icon, click triggers download

13. **`<ApplicationCaseStudy />`**
    - Card or full component for case studies
    - Props: projectName, image, description, productsUsed, link
    - "Read Full Case Study" CTA

14. **`<ContactForm />`**
    - Generic form component
    - Variants: General Contact, RFQ, Quick Enquiry
    - Props: formType, onSubmit, fields configuration
    - Includes validation, reCAPTCHA, loading states

15. **`<WhatsAppFloatingButton />`**
    - Fixed position (bottom-right)
    - Green WhatsApp icon with subtle animation
    - Click opens WhatsApp chat with pre-filled message

16. **`<NewsCard />`**
    - Used on News listing and "Latest News" sections
    - Props: image, title, excerpt, category, date, link

17. **`<Timeline />`**
    - Vertical timeline for company history, milestones
    - Props: events array (year, title, description)

18. **`<StatsStrip />`**
    - Horizontal strip with key statistics
    - Props: stats array (number, label, icon)
    - Count-up animation on scroll into view

19. **`<CertificationBadge />`**
    - Logo/icon for certifications and standards
    - Tooltip on hover with description
    - Props: badgeImage, name, description

20. **`<VideoEmbed />`**
    - Responsive video player (YouTube, Vimeo, or self-hosted)
    - Props: videoURL, thumbnail, autoplay, controls
    - Lazy load

21. **`<SearchBar />`**
    - Header search input
    - Auto-suggest dropdown (optional)
    - Props: onSearch, placeholder

22. **`<FilterPanel />`**
    - Used in Downloads Centre and Products listing
    - Multi-select dropdowns, checkboxes
    - Props: filters configuration, onFilterChange

23. **`<Pagination />`**
    - Page numbers, prev/next buttons
    - Props: currentPage, totalPages, onPageChange

24. **`<SocialShare />`**
    - Share buttons for LinkedIn, Facebook, Twitter, Email
    - Props: url, title, description

25. **`<CTASection />`**
    - Full-width call-to-action banner
    - Props: headline, description, buttons (RFQ, WhatsApp, etc.)
    - Background color/gradient variants

    - Tabbed interface for product detail pages (Overview, Specs, Installation, Chemical Resistance, Downloads)
    - Props: tabs array (label, content)

    - FAQ-style accordion
    - Props: items array (question, answer)

28. **`<ImageGallery />`**
    - Lightbox image gallery for product images

    - Prev/next navigation

29. **`<RelatedProductsCarousel />`**

30. **`<TestimonialCard />`** (optional)
